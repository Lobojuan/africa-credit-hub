## 1. إعداد بيئة الاختبار

### 1.1 معلومات النظام

| العنصر | التفاصيل |
|------|--------|
| التطبيق | Cross-Jurisdictional Central Data Hub & Credit Registry System v1.2 |
| قاعدة البيانات | PostgreSQL مع 21 جدولاً |
| أدوار المستخدمين | Admin, Regulator, Lender, Viewer |
| العملات المدعومة | أكثر من 42 عملة أفريقية بالإضافة إلى USD وEUR وGBP |
| التحسينات المؤسسية | MFA, Fuzzy Matching, Dispute Chatbot, OAuth 2.1, Low-Bandwidth, XBRL Upload, Tamper-Evident Audit, Exchange Rate Management, API Administration, Data Retention Policies, Global Search, ID Photos & Documents, Demo Environment |
| الولايات القضائية | جميع الدول الأفريقية الـ 54 |
| اللغات | الإنجليزية، الفرنسية، البرتغالية |
| البيانات التجريبية | أكثر من 102 ألف مقترض، أكثر من 172 ألف حساب ائتماني، 120 ألف سجل سداد، 3,218 نزاعاً، 2,147 حكماً قضائياً |
